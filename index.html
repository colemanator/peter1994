Portfolio

Prime Number Graph

This was a university Project created to demonstrate our ability to develop and deploy a scalable application to a cloud hosting service. The app takes a min and a max value and calculates the number of primes between those two numbers while also displaying the distribution of prime number on a Graph. The idea was to generate enough load through the prime number sift to educe configured automatic scaling. Although one of the initial reason for creating this app was it would generate load, it still uses a square root sift which increases efficacy. The app is built using node.js express and was designed to be as simple and minimalistic as possible.
